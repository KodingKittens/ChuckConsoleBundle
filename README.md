# ChuckConsoleBundle
Simple Symfony2 Bundle displaying a random Chuck Norris fact for every console command.

##Customization in config.yml file :
```yml
kk_labs_chuck_console:
    who:
        first_name: "Your first name"
        last_name: "Your last name"
```
